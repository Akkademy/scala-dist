IzPack führt Sie durch die Installation von Scala @VERSION@.

Diese Scala Distribution erfordert eine Java-Laufzeitumgebung (JRE) Version 1.6 oder neuer (eine 1.5-kompatible Distribution ist separat auf unserer Website erhältlich); JRE ist auch im JDK enthalten.

Sie können die Installation jederzeit abbrechen, indem Sie auf 'Beenden' klicken.
